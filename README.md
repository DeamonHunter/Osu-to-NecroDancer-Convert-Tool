Osu-to-NecroDancer-Convert-Tool
===============================

Converts .osu files to the NecroDancer Format. Info Can be found here http://braceyourselfgames.com/forums/viewtopic.php?f=5&t=314&p=3600#p3600

Created By DeamonHunter.
